Integração de Hardware com Raspberry Pi

Projeto acadêmico voltado ao desenvolvimento de um protótipo funcional de automação, utilizando Raspberry Pi e programação em Python.

O projeto envolveu a integração entre hardware e software, permitindo testes práticos de aplicações em Internet das Coisas (IoT), explorando comunicação entre dispositivos.

Durante o desenvolvimento, foram aplicados conceitos de sistemas embarcados, automação e controle de componentes físicos.

Essa experiência contribuiu para o desenvolvimento das minhas habilidades em integração entre software e hardware.

Tecnologias: Raspberry Pi, Python
