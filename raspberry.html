Integração de Hardware com Raspberry Pi

Projeto acadêmico voltado ao desenvolvimento de um protótipo funcional de automação, utilizando Raspberry Pi e programação em Python.

O projeto envolveu a integração entre hardware e software, permitindo testes práticos de aplicações em Internet das Coisas (IoT).

Durante o desenvolvimento, foram explorados conceitos de sistemas embarcados, comunicação entre dispositivos e automação de processos.

Essa experiência proporcionou aprendizado prático no desenvolvimento de soluções tecnológicas integradas e controle de dispositivos físicos por software.

Tecnologias: Raspberry Pi, Python
