Projeto Raspberry Pi

Este projeto foi desenvolvido com o objetivo de explorar conceitos de computação física e integração entre software e hardware utilizando a plataforma Raspberry Pi.

Durante o desenvolvimento foram aplicados conhecimentos de programação em Python para controle de dispositivos eletrônicos conectados ao microcomputador.

A proposta do projeto foi demonstrar como sistemas embarcados podem ser utilizados para automação e controle de dispositivos, permitindo a criação de soluções tecnológicas aplicáveis em diferentes contextos.

Esse tipo de aplicação é muito utilizado em áreas como automação residencial, Internet das Coisas (IoT) e sistemas inteligentes.

Tecnologias utilizadas:

  • Python
  • Raspberry Pi
  • Integração Hardware / Software
