Projeto AFA – UNICAP

O Projeto AFA é uma iniciativa voltada para a alfabetização de crianças que apresentam dificuldades no processo de aprendizagem da leitura e escrita.

O objetivo do projeto é oferecer apoio pedagógico por meio de atividades educacionais estruturadas que auxiliem no desenvolvimento das habilidades básicas de alfabetização.

Dentro desse contexto, foi desenvolvido um site institucional para apresentar o projeto, seus objetivos e suas atividades, além de servir como meio de divulgação da iniciativa.

O desenvolvimento foi realizado utilizando tecnologias de desenvolvimento web, com foco em uma interface simples, clara e acessível.


Tecnologias utilizadas

  • CSS
  • Vercel
