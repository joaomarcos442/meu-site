Projeto COLISEU – LICEU (ArcheoMap)

Projeto acadêmico desenvolvido em equipe com foco na criação de um sistema web institucional, voltado à organização e apresentação de informações.

Atuei no desenvolvimento front-end, sendo responsável pela construção da interface, estruturação das páginas e organização visual da aplicação.

Foram aplicados conceitos de design, responsividade e usabilidade, buscando proporcionar uma navegação intuitiva.

O projeto contribuiu para o desenvolvimento de habilidades em trabalho em equipe e construção de aplicações web.

Tecnologias: HTML, CSS
