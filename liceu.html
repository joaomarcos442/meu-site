Projeto LICEU

O Projeto LICEU foi desenvolvido no contexto acadêmico com o objetivo de aplicar na prática conceitos de desenvolvimento de interfaces web.

Durante o projeto foram trabalhados princípios de estruturação de páginas, organização de conteúdo e estilização utilizando tecnologias de front-end.

A proposta foi construir uma interface clara e funcional, permitindo apresentar informações de forma organizada e visualmente agradável.

Esse projeto contribuiu para o desenvolvimento de habilidades relacionadas à criação de layouts responsivos e boas práticas de desenvolvimento web.

Tecnologias utilizadas:

  • HTML
  • CSS
