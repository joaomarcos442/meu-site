Sobre o Projeto

O Projeto Liceu consiste no desenvolvimento de um site educacional voltado para organização e apresentação de conteúdos acadêmicos. A ideia principal foi criar uma interface simples e intuitiva que facilite o acesso às informações e materiais disponíveis para estudantes e professores. Esse projeto permitiu praticar conceitos de estruturação de páginas web utilizando HTML e estilização com CSS.
