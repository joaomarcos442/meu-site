Projeto COLISEU – LICEU (ArcheoMap)

Projeto acadêmico desenvolvido em equipe com foco na criação de um sistema web institucional.

Atuei no desenvolvimento front-end, sendo responsável pela construção da interface e organização visual da aplicação, buscando oferecer uma navegação intuitiva e uma boa experiência ao usuário.

Foram aplicados conceitos de design, estruturação de páginas e usabilidade, garantindo uma interface clara e funcional.

O projeto contribuiu para o desenvolvimento de habilidades em trabalho em equipe e construção de interfaces web.

Tecnologias: HTML, CSS
